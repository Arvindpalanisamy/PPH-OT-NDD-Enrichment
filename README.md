# PPH-OT-NDD-Enrichment
NDD risk gene enrichment analysis for mPFC and hypothalamus DEGs following postnatal oxytocin exposure
## Description
Fisher's exact test enrichment of neurodevelopmental disorder (NDD) 
risk genes among differentially expressed genes (DEGs) from mPFC and 
hypothalamus following postnatal oxytocin exposure in rat offspring.
## Reference databases
- SFARI Gene (January 2026 release)
- Developmental Brain Disorders (DBD)
- SysNDD (Definitive tier, accessed March 2026)
## Requirements
Python 3.x
pip install openpyxl scipy pandas
## Usage
Place DEG files in ./data/ directory (see script header for filenames)
python ndd_enrichment_analysis.py
